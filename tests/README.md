Tests placeholders

- Archivo: `tests/calculate.test.ts` contiene `it.todo` para las funciones en `src/logic/calculate.ts`.

Sugerencia rápida para ejecutar tests (no instalado por defecto):

1. Instalar Vitest (ejemplo con pnpm):

```bash
pnpm add -D vitest @types/node
```

2. Ejecutar:

```bash
pnpm vitest
```

Puedes adaptar a Jest si lo prefieres.
